# javascriptの練習を兼ねてバスケのスコアリングアプリを作る

  ## 作りたい機能

    * away,homeに分かれたチームごとの得点表示
      * スクロールしても固定
    * 1,2,3点を加算できるボタン
    * 出場選手の番号、名前(左右にチームごとサイドバーにする)
      * 選手の番号、名前の登録追加 
    * 誰が得点したか記録し、履歴を表示
      * 古い順に得点表示の下に追加されていく
      * スクロールすると”一番上に戻る”ボタン出る
    * シュート確率を自動計算
      * シュート投下カウント→成功(boolean)で得点カウント
      * 選手ごと、チームごとで計数
